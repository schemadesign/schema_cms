# Jobs

## Get job data

**Request**:

`GET` `/api/v1/jobs/:id`

*Note:*

- **[Authorization Protected](authentication.md)**

**Response**:

```json
Content-Type application/json
200 OK
{
    "pk": 8,
    "datasource": 8,
    "steps": [{"script":  1, "exec_order":  0}],
    "job_state": "success",
    "result": "http://localhost:8000/schemacms/storage/projects/1/datasources/8/results_test.csv",
    "error": ""
}
```
